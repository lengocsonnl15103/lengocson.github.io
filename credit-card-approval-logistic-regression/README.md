# Predicting Credit Card Approvals Using Logistic Regression

## Project Overview
This project analyzes applicant data to predict **credit card approval outcomes** using Logistic Regression.
The goal is to demonstrate how data analysis and statistical modeling can support **credit decision making** in banking and financial institutions.

---

## Business Context
Credit card approval decisions directly impact:
- Credit risk
- Customer acquisition
- Profitability of financial institutions

This project addresses the key question:
**Can we predict whether a credit card application should be approved based on applicant characteristics?**

---

## Objectives
- Explore and understand credit card application data
- Preprocess data for modeling
- Apply logistic regression for binary classification
- Interpret results from a business and risk perspective

---

## Methods & Techniques
- Exploratory Data Analysis (EDA)
- Data preprocessing and feature encoding
- Logistic Regression modeling
- Model evaluation and interpretation
