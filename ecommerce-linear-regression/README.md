# Ecommerce Analysis using Linear Regression

## Project Overview
This project analyzes ecommerce data to understand the relationship between key variables and business performance.
Linear Regression is applied to identify how different factors influence the target metric and to support data-driven decision making.

---

## Business Problem
Ecommerce companies need to understand which factors most strongly impact performance metrics such as revenue, customer behavior, or sales outcomes.

This project addresses the question:
**Which variables have the greatest influence on ecommerce performance?**

---

## Objectives
- Explore and understand ecommerce data
- Identify relationships between variables
- Apply linear regression to model these relationships
- Interpret results from a business perspective

---

## Methods & Techniques
- Exploratory Data Analysis (EDA)
- Data visualization
- Linear Regression modeling
- Model evaluation and interpretation
