# Bank Debt Recovery Analysis

## Project Overview
This project analyzes bank loan and customer data to support **debt recovery and credit risk management**.  
The analysis applies statistical methods, machine learning, and time-series forecasting to extract actionable insights for banking operations.

---

## Objectives
- Analyze customer financial behavior related to loans and repayments  
- Segment customers based on risk characteristics  
- Identify key factors affecting repayment performance  
- Forecast debt recovery cash inflows for financial planning  

---

## Methods & Techniques
- Exploratory Data Analysis (EDA)
- K-means Clustering for customer segmentation
- Principal Component Analysis (PCA)
- OLS Regression for relationship analysis
- ARIMA Time-Series Forecasting
